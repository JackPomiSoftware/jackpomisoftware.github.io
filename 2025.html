2025 Jack Pomi Software Report

The year 2025 was a hard one, with many releases or updates had to be postponed. If you want a reason - well, it is some real life problems and time-consuming tasks that were incompatible with software development. Jack Pomi Software is a solo project, after all, so this situation was predictable, but of course annoying and unpleasant as well. But let's see how did Jack Pomi software progress this year, because it definetely did some major steps in 2025.

On 1 February, third minor software title was released - IsGenuine. It allows user to check the activation status of your operating system.

This was soon followed by one major leap for the project. On 29 July, Ultimate Control was released. This is a first large-scale software project released by Jack Pomi Software. This software allows you to change settings of your Windows operating system. Redmond tried to move settings from Control Panel to new Settings app since 2012 and...still trying to do it. Maybe I will do it faster than them? We'll see... Watch out for a new update soon!

IE Return was augmented with a feature backport from Ultimate Control. Now you can set up multiple home pages with it. Also, in December, this program received first bug issue on GitHub. This became the start of a feedback service by Jack Pomi Software. If only I haven't noticed that a month later...

And what about the future?

I hope that the year 2026 will bring more positive changes and achievments than the previous year. And I will try my best to make it a reality. See you next year!

Report made by Jack Jackie Pomi, 11 January 2025.
