Imagine a web like 20 years ago

What is this all about?

Today, the absolute majority of websites use the flat user interface design. Some call it Metro, some call it Fluent, but no matter how you slice it — it's still that oversimplified UI that takes standartisation as a primary, but not personalisation or visual appeal. Web2009 will be a series of web services that are aimed on restoring the mid-late 2000's skeumorph look of websites. But of course, technologies advance rapidly, so Web2009 will of course utilise modern abilities of websites that were introduced after the 2000s decade.
