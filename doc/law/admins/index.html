Administrators and standard users

Ever since user accounts were introducedAlthough user accounts have existed since Windows 95, they were greatly improved in Windows XP, which introduced differentiation of user accounts into standard user accounts, administrators and guests, as well as user-friendly logon screen. in Windows XP, they were divided into three large groups: standard users, administrators and guests. By default, Windows allows you to create standard user accounts and administrators accounts, and a single built-in guest account is disabled by default.

  • Standard user accounts are capable of installing and running most programs as well as changing a limited number of the system settings, particularly those unrelated to security or to other user accounts. They are often used for children or in offices for non-administrator employees. Usually, in the former case, the usage of standard user account is accompanied by the installation of parental controlsParental controls is a software that helps parents control their children's computer usage. It usually includes a blacklist of websites, restrictions on using particular software and games and usage time restrictions.. A feature that controls the limiting of functionality is User Account ControlA User Account Control (UAC) is a Windows feature that prevents users from making unwanted changes to computer or from performing forbidden actions. By default, when UAC window appears, the screen darkens and all other windows temporarily dissappear..
  • Administrator accounts enjoy full access to the computer's settings and can install and run any software with no restrictions. When you install Windows, you are forced to make an administrator account. At least one administrator account must exist in the system.
  • Guest accounts are very limited in their functionality: they can not install new software, change any settings or create a password. Every time a guest account logs out, the whole former session is deleted and no previous files are saved. By default, guest user account is disabled by the system. Guest accounts are used on public computers in places like libraries.
What user account type should I choose?

That depends on purposes you are creating new user account for. For example, if you are creating an account for your child, choose standard user account.
