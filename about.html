About Jack Pomi Software

Jack Pomi Software is a small, or to be precise, a solo software organisation created by Jack Jackie Pomi. I create small software and even games.

The organisation came into existense on 12 February 2023, when I released SRDC racing game. Since then, the organisation continues to expand, switching it's focus towards computer software, primarly for Windows operating systems. The first piece of software, IE Return, was released on 4 February 2024. This website opened six days later.

All software made by Jack Pomi Software is published on GitHub. Games are released on itch.io.

This website is inspired by "windows.microsoft.com" website as it looked from 4 August 2009 to 27 June 2012.
